# Report Style Hint
- **Selected style**: standard
- **Suggested headers**: 概要, 現状, 次のアクション

## Notes
目的・現状・次のアクションを含む標準的な報告。必要に応じて箇条書きを使用。
NarrativeGenプロジェクトでは、Unity C#開発に関する技術的な詳細も含める。

## Banned phrases
- 了解しました
- 続けてください
- 特にありません
- 完了しました
- 向上されました
- 問題ありません
- 実装しました（具体的な内容を記載）
- 修正しました（何を修正したか明記）
- 追加しました（何を追加したか明記）
- 削除しました（何を削除したか明記）
- Unity関連の用語を誤用（例: "GameObject"を"ゲームオブジェクト"と表記しない、適切なUnity用語を使用）

## Reminder
- Include purpose, current status, and next actions in a way that fits the selected style.
