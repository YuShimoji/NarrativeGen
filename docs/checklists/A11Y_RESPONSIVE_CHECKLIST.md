# A11Y / Responsive Checklist

最終更新: 2026-04-08

対象画面: Story / Graph / Play / Modal

## 共通チェック

- [ ] キーボードのみで主要操作が完結する
- [ ] フォーカス順が視覚順と一致する
- [ ] フォーカス可視化が常に確認できる
- [ ] スクリーンリーダーで役割とラベルが読める
- [ ] コントラスト比が不足しない
- [ ] 文字拡大（200%）時に操作不能にならない

## Story View

- [ ] 選択肢ボタンに意味のあるラベルが付いている
- [ ] 状態表示（flags/resources/variables）が見出し構造を持つ
- [ ] モバイル幅でテキストと選択肢が重ならない

## Graph Editor

- [ ] 主要操作にキーボード代替経路がある
- [ ] ノード選択状態が視覚/音声で判別できる
- [ ] タブレット幅でパネル崩れが発生しない

## Play View

- [ ] モード切替 UI がキーボード操作可能
- [ ] 画像/BGM状態が操作不能を起こさない
- [ ] 小画面で本文・選択肢・操作導線が維持される

## Modal

- [ ] 開いた時にモーダル内へフォーカスが移る
- [ ] 閉じるまで背面へフォーカスが逃げない
- [ ] Escape で閉じられる（破壊的操作は確認あり）

## テスト責務境界

- 自動化（E2E）:
  - [ ] 表示・遷移・主要クリックフロー
  - [ ] 回帰しやすい仕様（モーダル開閉、タブ遷移、保存導線）
- 手動確認:
  - [ ] スクリーンリーダー読み上げ品質
  - [ ] 聴覚/体感品質（BGM クロスフェード）
  - [ ] 実機モバイル操作感（タップ領域、横持ち/縦持ち）
