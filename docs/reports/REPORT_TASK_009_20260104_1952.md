WORKER_PROMPTの停止条件「ブラウザ操作が不可能な環境（手動テストのため）」に該当します。

- MCPリソース（cursor-browser-extension、cursor-ide-browser）が利用できない
- 手動テストのため、実際のブラウザ操作が必要

3. **GUI編集モードに入る**

4. **テストケースの実施**
