使用方法

1. サンプルモデルを読み込む
11個のサンプルノードを内部に読み込みます。

2. キーワードを入力
検索ボックスに検索したい言葉を入力します。

3. 同義語検索
チェックボックスをオンにすると、「battle」で「fight」「combat」「戦闘」など関連語も検索します。

4. 検索例をクリック
右側の検索例から例を選択すると、自動で検索が実行されます。
